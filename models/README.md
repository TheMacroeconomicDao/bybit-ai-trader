# ML Models Storage

Директория для хранения обученных ML моделей для торговой системы.

## Модели

- `pattern_success_rf.joblib` - Random Forest модель для предсказания успешности паттернов (создается автоматически после обучения)

## Использование

Модели создаются и сохраняются автоматически модулем `ml_predictor.py` после накопления достаточного количества данных для обучения.
